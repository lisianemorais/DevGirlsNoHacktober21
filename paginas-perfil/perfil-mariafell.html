Um pouco sobre mim

Sou a Malu, atualmente desenvolvedora front-end. Fiz transição de carreira em 2020, estava com 30 anos e há mais de 10 no mercado de trabalho, então foi uma aventura que iniciou cheia de incertezas, mas terminou em uma grande realização :)
Além de TI, na minha trajetória profissional atuei com gestão de processos e melhoria contínua, passei por indústrias e varejo trabalhando em projetos com metodologias lean, BPM e ágil.

Informações Profissionais

  • 2022 - atual: Front-end Engineer Jr - Intelipost

    • Após a fusão da Pegaki com o grupo Intelipost, passei a trabalhar com um novo produto atuando na implementação de features com Javascript, Vue.js 3 e Vite, Vuetify, HTML, CSS e SASS, Pinia, consumo de APIs com URQL, implementação de testes unitários com Vitest e Vue Testing Library, Storybook para documentação dos componentes.

  • 2021 - 2022: Front-end Engineer Jr - Pegaki Soluções Logísticas

    • Atuei na startup Pegaki, com a implementação de features utilizando Javascript, Vue.js 2, Vuetify, HTML5, CSS3 e SASS, Vuex, consumo de APIs com Axios, implementação de testes automatizados com Cypress, versionamento com GIT.

  • 2020 - 2021: Transição de Carreira

    • Em 2020 iniciei transição de carreira para TI, foquei meus estudos em desenvolvimento web com foco em Javascript. No backend estudei Node.js com Express, Postgres, ORM Sequelize, API Rest, Postman, testes unitários com Jest. No frontend estudei HTML, CSS com Flex, Bootstrap, EJS como template engine, um pouco de React. Subia meus projetos no Heroku, Vercel e Netlify.
